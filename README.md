# 🪐 Asteroids (Boot.dev Project)

This is a basic Python implementation of the classic arcade game **Asteroids**, created as part of the [Boot.dev](https://boot.dev) backend development course.

## 🎯 Goal

Build a simple game using Python and object-oriented programming (OOP) concepts:
- Classes for player, asteroids, and shots
- Basic collision detection
- Modular code structure
